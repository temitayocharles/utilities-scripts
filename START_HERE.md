# Start Here

This repo stores utility scripts used across projects (scanners, backups, helpers).

## 1. What This Repo Is
Central collection of operational scripts for scanning repos, managing configs, and automation.

## 2. Repo Map (What Lives Where)
- `scripts/`: executable tools.
- `tests/`: unit tests for scripts (where applicable).
- `docs/`: usage docs for each script.

## 3. How to Use
1. Read the script’s doc in `docs/`.
1. Run with `python <script>` or `bash <script>` as documented.
1. Review output files before applying changes.

## 4. Troubleshooting
- Ensure dependencies are installed (see each script’s doc).
